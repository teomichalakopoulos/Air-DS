# Air-DS
This project is a web application for the ficitonal airline “Air DS”. The application was designed to allow users to easily and quickly book airline tickets, as well as view and manage their reservations. Technologies used: HTML5, CSS3, JavaScript, and PHP, with a MySQL database.

The project is meant to be used using the XAMPP tool. The pdf: AIR_DS_Website/WEB_ΜΙΧΑΛΑΚΟΠΟΥΛΟΣ_ΘΕΟΔΩΡΟΣ_Ε22108.pdf explains how the website works.
